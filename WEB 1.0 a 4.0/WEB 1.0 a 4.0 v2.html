WEB 1.0, 2.0, 3.0 y 4.0


WEB 1.0

La Web 1.0 se refiere a la primera etapa de la evolución de la World Wide Web. La web 1.0 es un tipo de web estática con documentos que jamás se actualizaban y los contenidos eran dirigidos a la navegación HTML y GIF.
La web 1.0 apareció en el año de 1990 con HTML como lenguaje hipertexto con el objetivo que las páginas Web sean más agradables a la vista con componentes como imágenes, vídeo y colores. Los cuatro elementos esenciales del diseño de un sitio web 1.0 son:

  • Paginas estaticas
  • El contenido se sirve desde el sistema de archivos del servidor
  • Paginas creadas con las inclusiones del lado del servidor o la interfaz de puerta de enlace comun.
  • Marcos y tablas utilizados para colocar y alinear los elementos en una pagina

WEB 2.0

La web 2.0 se refiere a la transición de las webs tradicionales a webs destinadas a usuarios.
Es una web orientada a facilitar la interacción entre los usuarios y el desarrollo de redes sociales, donde pueden expresar y opinar, administrando su propio contenido. Permitiendo realizar envíos, búsquedas, recibir información de interés, colaborar con otros usuarios y crear conocimiento.
Algunas de las caracteristicas son:

  • Páginas dinámicas, integran recursos multimedia como: videos y sonidos.
  • Javascript, lenguaje utilizado para diseñar los formatos de las páginas.
  • Empleo de interfaces de fácil entendimiento para la interacción del usuario.
  • Permite al usuario crear y administrar su propio contenido.
  • La información se transmite unidireccional o bidireccional.

WEB 3.0

Web 3.0, son aplicaciones web conectadas a aplicaciones web, a fin de enriquecer la experiencia de las personas; a esto agrega conocimiento del contexto en la web geoespacial, la autonomía respecto del navegador y la construcción de la web semántica.
El término web 3.0 apareció por primera vez en 2006 en un artículo de Zeldman, crítico de la web 2.0. Esta web fue operativa desde el 2010. La web 3.0, es un salto tecnológico que tiene importantes consecuencias en los usuarios de la red.
La web 3.0 es conocida como la “web semántica” porque utiliza de forma más eficiente de los datos: “data web”. Es inter-operativa y el usuario tiene el control para hacer los cambios que desee modificando directamente las bases de datos.

WEB 4.0

La web 4.0 se enfoca primordialmente en la experiencia de usuario al diseñar o crear plataformas, productos o servicios. Esta se trata de una web activa que funciona como un asistente virtual con AI, la utilización del Big data, el machine learning, VR y IoT para una maxima personalizacion y optimizacion de tareas.
Esta también se conoce como web simbiótica, donde su propósito es la interacción simbiótica entre el hombre y la máquina.
El objetivo de esta es mantener el sistema siempre en línea, de alguna manera como un sistema operativo web con información de todos lados, en un solo lugar.
